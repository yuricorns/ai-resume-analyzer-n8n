# AI Resume Analyzer (ATS Skill Matching)

An AI-powered resume analyzer that compares a resume with a job description, calculates an ATS-style skill match percentage, identifies missing skills, and generates resume improvement suggestions.

##  Features
- Resume vs Job Description skill comparison
- ATS-style skill match percentage
- Missing skill identification
- AI-generated resume improvement suggestions
- ATS optimization tips
- Multi-agent LLM workflow

##  How It Works
1. User submits resume text and job description
2. AI Agent extracts skills in structured JSON format
3. Matching logic calculates skill overlap and percentage
4. AI Agent generates human-readable resume feedback

##  Tech Stack
- n8n (workflow automation)
- LLM (Groq / OpenAI compatible)
- JavaScript (business logic)
- Prompt Engineering
- AI Agents

##  Use Case
- Job seekers optimizing resumes
- ATS keyword gap analysis
- Career guidance using AI

##  Learnings
- Handling structured vs unstructured LLM outputs
- Designing multi-agent AI workflows
- Defensive JSON parsing
- Real-world AI debugging
